# Homebrew Tap for claude_pro

This tap provides the Homebrew formula for [claude_pro](https://github.com/ilya-savinkov/ClaudePro) — a CLI tool for running Claude Code in a non-interactive loop.

## Installation

```bash
brew tap ilya-savinkov/claude_pro
brew install claude_pro
```

## Requirements

- Ruby >= 4.0
- [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code) >= 2.1.76

## Updates

This formula is automatically updated by the [ClaudePro release workflow](https://github.com/ilya-savinkov/ClaudePro/blob/main/.github/workflows/release.yml) when a new version is published.
